## 你要做的四步

### Step 1 — 研究：找同格死者
研究问题：**2022–2025 年，押注「AI agent / LLM 驱动的垂直任务自动化」、拿过融资或有明显声量、但已死亡或明显衰败的公司有哪些？**

死亡/衰败的判定（满足其一即可入墓碑）：
- 已关停 / 被贱卖（acqui-hire 价低于总融资）
- ARR 萎缩或长期停滞、估值腰斩
- 创始团队公开宣布转型放弃原方向

每个候选要查清并记录：
- 公司名、国家、成立年、累计融资、巅峰估值/ARR
- **死法的一句话机制**（这是关键——不是「失败了」，是「为什么这个结构位置撑不住」。例：被底层模型一次升级直接吞掉 / 单点功能没护城河 / demo 惊艳但留存崩 / 获客成本高于 LTV）
- 死亡/衰败时间
- 证据来源 + 证据等级（A 一手财报/官宣 / B 权威媒体 / C 第三方估算/传闻 / X 仅传言）

数量要求：**至少 5 个，其中 ≥3 个要做到 B 级以上证据。** 不足 5 个要说明搜索过程，不要凑数编造（编造对标=违反 SK/本项目铁律）。

中文世界同类也要扫一遍（中国押 agent 垂直、已死的）——这关系到「中国读数」是否准确。

### Step 2 — 写入墓碑栏

算出并写明 **生死比（活 N / 死 M）**。

### Step 3 — 据生死比修正判断
墓碑补全后，回头看这两栏，按事实修正（不要为了好看而保留乐观结论）：
- `## 邻近空格`：如果死者集中暴露了某种结构性死法（比如「被底层模型吞掉」反复出现），那么相邻的中国空格 / L4 空格还值不值得点亮？据此调整措辞。
- `## 机制提炼`：把死者们的共同死法，提炼成一条可迁移机制（一句话）。这是本格留给其它格子的复利资产。
