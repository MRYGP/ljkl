# TASK-007｜主动环境辅助·残余空位证据补齐（冲第一条活预测）

> 版本：v1.0 · 2026-06-02
> 性质：单靶深挖。只钻一个空位，不横扫别的格。目标是产出本档案**第一条扛得住对账的活预测**——或者精确说清还差哪条腿，绝不硬凑。
> 来源：TASK-006 §B1 从 ORPHAN-001（Google Glass）剥出的严格差集。
> 上位约束：先读 `00-宪章.md`、`坐标系.md`、`AGENTS.md`。三铁律违反即停。

## 靶心（唯一研究对象）
> **无需显式唤起、可全天佩戴、隐私边界可被社会接受的主动环境辅助。**

注意边界：已点亮的是「按需视觉 AI」（说「Hey Meta」/抬手/捏合后给答案）。本靶是它的**残余补集**——不靠显式唤起、持续在背景里、且旁人能接受的主动辅助。任何已上市的「按请求」能力都不算填了这个格。

## 硬门槛（每一步都适用）
- **先做「信号是否已被满足」体检（最高优先）。** P-003 就是死在这——立项时验证信号已被覆盖。动手前先查：截至今天，有没有产品已经做到「无需显式唤起 + 全天主动 + 隐私可接受」三者同时成立？
  - 全部成立 → 这格已点亮，**不立预测**，记为「已填/失格」，写明谁、何时、证据，收工。
  - 部分成立 → 把仍空的那一小块精确描述成预测靶，验证信号必须能区分「主动无唤起」与「按需」。
- **踏脚石可定日、可引用。** 让这格此刻可能的是什么刚翻转——候选方向：端侧模型、续航、EMG/神经带等低调输入、隐私友好采集、监管口径。每个给翻转点 + 日期 + 来源。给不出就老实说没有，不准用「邻近可见」搪塞。
- **无墓碑不入档（铁律二）。** 必须有生死比。Humane Ai Pin 已在档（近邻）。再扫这一波「AI 随身/吊坠/眼镜想做常驻助手」的死者（如 Rewind/Limitless、Friend、Bee、Plaud、Tab 等，自行核实，不照抄此清单），每个标口径（同格/近邻），算活 N / 死 M。
- **象限由死法定，不预设。** Glass 死法已指向「社会接受度/隐私锁主导，兼技术锁」。本靶需求是**明牌**（人公开想要随身助手），门是**隐私/社会接受度锁** → 大概率落 **左下（明牌×社会/羞耻锁）**。
  - 左下**不是**红线（红线只在右下 暗牌×羞耻锁）。所以本格**保留全部死者名、活公司名与来源作为证据，不要匿名**。（接上一轮 CELL-026 的教训：红线删的是可复制利用细节，不是可核查身份；本格根本不在红线内，更不该匿名。）
  - 若死法核查后象限与上述不符，按证据改并记一行修订（原→新→依据→日期），别将就。
- **未找到是合法结果。** 不造案例、不为出预测升证据级。
- **不功利（铁律三）。** 不出现变现、增长、回报期限。
- 证据 A/B/C/X，X 仅灵感。编辑只动本任务触发的部分。

## 要补齐的三条腿（齐了才够立预测）
1. **第三方需求痕迹**：这个「不想每次唤起、想要常驻主动辅助」的需求是否一直存在、有第三方痕迹（抱怨/影子用法/反复弃用唤起词）。不是「我觉得应该有」。
2. **可定日踏脚石**：哪个具体能力/成本/形态/监管刚翻转，让全天 + 无唤起 + 隐私可接受第一次可能。
3. **同格/近邻墓碑 + 生死比**：谁押注常驻主动辅助而死，共同死法是什么（多半正是社会接受度/隐私/续航）。

## 产出（二选一，按证据决定，不得凑数）
**A 够条件 → 立 P-004，写进 `空格子-预测库.md`，严格按文件顶部写法规约：**
- 坐标（层 × 地域 × 象限，按死法定）、踏脚石（带翻转点+日期+来源）、相邻格状态（活/墓碑列名，生死比）、门外的人（具体）、预测（**时间窗 + 可观测且当前尚未为真的验证信号**）、证据级、下次对账日、立此日期。
- 验证信号自检：写完问一句「这个信号今天是不是已经为真？」是 → 改信号或降级，别重蹈 P-003。
- 同步：若需要可建一个 `status: open/active` 的 CELL 文件承载本格（按 `_模板.md`），预测库版本号 +0.1。

**B 缺任一条腿 → 不立预测。** 在 `产出-TASK-007-*.md` 写清：哪条腿缺、缺到什么程度、补什么证据能补上；把已找到的痕迹/翻转点/墓碑都落档，供下轮直接接力。这同样是合格产出。
