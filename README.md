# 邻近可能（ljkl）

> 纯学术研究档案。研究「伟大」从哪些结构位置长出，并预测当下结构上本该长出、却还空着的位置。

## 一句话
> 伟大不能被**计划**，但能被**照亮**。这台引擎不造伟大，它照亮伟大可能出现的格子，提高你撞见它的概率。
